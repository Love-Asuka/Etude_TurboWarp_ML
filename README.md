# Etude-TurboWarp-ML

一个为 TurboWarp 平台开发的机器学习扩展，允许用户在可视化编程环境中构建、训练和部署简单的神经网络模型。

## 功能特性

### 核心模块
- 模型定义：支持创建具有多个线性层的神经网络
- 前向传播：执行模型推理并获得预测结果
- 模型管理：查看模型结构、清除模型等操作

### 自动微分模块
- 反向传播：基于计算图的自动微分系统
- 梯度获取：访问特定层的梯度信息
- 梯度清零：重置所有梯度值

### 优化器模块
- SGD优化：随机梯度下降算法更新模型参数
- 学习率调整：动态设置优化过程的学习率

### 线性代数工具
- 矩阵乘法：执行两个矩阵的乘法运算
- 矩阵加法：执行两个矩阵的加法运算

## 安装与使用

### 在 TurboWarp 中加载扩展

1. 打开 [TurboWarp](https://turbowarp.org/)
2. 点击"添加扩展"按钮
3. 选择"从文件加载扩展"
4. 选择本项目的 `main.js` 文件

### 积木说明

#### 核心模块积木

1. **开始模型定义**
   - 参数：输入维度（数字）
   - 作用：初始化一个新的神经网络模型，指定输入数据的维度

2. **添加线性层**
   - 参数：
     - 输出维度（数字）：该层输出的维度
     - 激活函数（下拉菜单）：可选 ReLU、Tanh、Sigmoid、Softmax 或无激活函数
   - 作用：向模型中添加一个线性变换层

3. **结束模型定义**
   - 作用：完成模型构建阶段，初始化模型参数

4. **推理**
   - 参数：输入向量（JSON格式的二维数组）
   - 返回：模型预测结果（JSON格式的二维数组）

5. **获取模型结构JSON**
   - 返回：包含完整模型信息的JSON字符串

6. **清除模型**
   - 作用：重置整个模型状态

7. **模型是否已定义**
   - 返回：布尔值，表示模型是否已完成定义

#### 自动微分模块积木

1. **对输出梯度执行反向传播**
   - 参数：输出梯度（JSON格式的二维数组）
   - 作用：根据给定的输出梯度计算所有参数的梯度

2. **获取参数的梯度**
   - 参数：参数名称（下拉菜单）
   - 返回：指定参数的梯度值（JSON格式）

3. **清零所有梯度**
   - 作用：将所有参数的梯度重置为零

#### 优化器模块积木

1. **设置学习率**
   - 参数：学习率（数字）
   - 作用：设置SGD优化器的学习率超参数

2. **执行SGD优化步骤**
   - 作用：使用当前梯度更新模型参数

#### 线性代数工具积木

1. **矩阵乘法**
   - 参数：矩阵A和矩阵B（均为JSON格式的二维数组）
   - 返回：两矩阵相乘的结果

2. **矩阵加法**
   - 参数：矩阵A和矩阵B（均为JSON格式的二维数组）
   - 返回：两矩阵相加的结果

## 使用示例

以下是一个完整的神经网络训练流程示例：

1. 使用"开始模型定义"积木，设置输入维度
2. 使用"添加线性层"积木，添加若干隐藏层
3. 使用"结束模型定义"积木，完成模型构建
4. 准备训练数据
5. 执行前向传播得到预测结果
6. 计算损失函数相对于输出的梯度
7. 使用"对输出梯度执行反向传播"积木计算参数梯度
8. 使用"执行SGD优化步骤"积木更新模型参数
9. 重复步骤5-8进行多轮训练

## 技术架构

扩展由以下几个核心组件构成：

1. **EtudeTurboWarpMLCore**：负责模型定义、前向传播等核心功能
2. **EtudeTurboWarpMLAutograd**：实现自动微分系统
3. **EtudeTurboWarpMLOptimizer**：提供参数优化算法
4. **EtudeTurboWarpMLLinearAlgebra**：基础线性代数运算工具
5. **MLUtils**：通用数学工具函数集合

## 支持的激活函数

- ReLU：`f(x) = max(0, x)`
- Tanh：`f(x) = tanh(x)`
- Sigmoid：`f(x) = 1/(1+exp(-x))`
- Softmax：`f(x_i) = exp(x_i)/Σ(exp(x_j))`
- None：无激活函数（恒等映射）

## 注意事项

1. 所有矩阵运算均使用JSON格式的二维数组表示
2. 模型必须先完成定义才能执行推理或训练操作
3. 进行反向传播前应确保已完成前向传播
4. 每次优化步骤后建议清零梯度以避免累积
